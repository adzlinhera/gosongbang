AGC generate contents for your blog, based on your desired keyword.

== Description ==

You can have AGC blog, generated by scraping Bing search (images and text), based on your desired keyword.

Major features in VM AGC include:

* Configuration Page.
* Image search, with size filter. And text scrape.
* Allow you to specify the image dimension for display.
* Allow you to specify to show your AdSense only to visitors (refered) from search engines.
* Generated Related Posts to link for other post (actually, keyword).
* Allow you to setting your own agc template


== Installation ==

Upload the VM AGC plugin to your blog, Activate it, then go to configuration page.
Setup all the values and conditions.
To trigger it, paste this to your index or home template before `if ( have_posts() ) section
    <?php vm_agc_before_content( ); ?>
Make sure to comment out "No Post Found" section on your index.php or home.php or overwrite it with the script above.
	
1, 2, 3: You're ready!
